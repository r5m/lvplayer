Untitled player

How does it work?

  1. Open video file

    • Supported video formats are the same as for standard "Video" app

  2. Optionally add subtitles or external audio track

    • Only srt subtitles supported.
    • Subtitles must be in .TXT file (not .srt).
    • You may need additional application from a marketplace to open .txt file. Something like "Explorer" or "Finder"
    • Supported audio formats are the same as for standard "Music" app
